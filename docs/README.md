# Documentation Index

Use these guides when cloning, rebranding, deploying, or handing off the template:

- `SETUP_GUIDE.md`: local setup, Supabase setup, and first-run verification
- `BRANDING_GUIDE.md`: brand config, colors, fonts, contact info, and restaurant-type adaptation
- `CONTENT_GUIDE.md`: menu, specials, hours, homepage, gallery, and owner editing workflow
- `DEPLOYMENT_GUIDE.md`: local preflight, Vercel deployment, production envs, and validation
- `HANDOFF_GUIDE.md`: client onboarding, launch checklist, asset collection, and maintenance workflow

Recommended reading order for a new developer:
